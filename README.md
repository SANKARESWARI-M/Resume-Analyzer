# Resume Analyzer & Job Matcher

## Overview
The Resume Analyzer & Job Matcher is a Python-based application that evaluates how well a resume matches a given job description. It extracts relevant keywords using Natural Language Processing (NLP) techniques and provides a match score along with missing skill suggestions.

## Features
- Extract text from PDF resumes  
- Perform keyword extraction using NLP (NLTK)  
- Compare resume content with job description  
- Generate a match score (percentage)  
- Identify matched keywords  
- Suggest missing keywords  
- Command-line interface for user interaction  

## Tech Stack
- Python  
- NLTK (Natural Language Processing)  
- PyPDF2 (PDF text extraction)
